# Viralizador — DevRel & Open Source GTM Program

## Summary
Recurring content creation program to generate viral short-form video clips showcasing Genie's capabilities, targeting developer communities on X/Twitter, Reddit, and LinkedIn.

## Status
Brainstorming phase. First deliverable: 15-second viral video.

## Creative Direction
- 1990s Shell F1 ads aesthetic — orchestral, epic, cinematic grandeur
- Opera/dramatic music over fast cuts of real agent activity
- Tone: provocative confidence, manifesto energy, FOMO-as-art
- "Something massive is being built and you're already late"
- No sales pitch — philosophical provocation

## Key Angles Identified
1. **"Wishes in, PRs out"** — the core promise, demonstrable in seconds
2. **The Numbers** — 48 PRs/week, 0.7h merge time, 96% SHIP rate
3. **Overnight Mode** — "Go to bed. Wake up to reviewed PRs."
4. **Context Collapse** — 7 tabs, 7 agents, zero coordination → Genie fixes this
5. **BYOA** — Not vendor locked. Claude, Codex, anything.

## Tools
- Descript (video editor with AI narration, captions, layouts)
- Raw screen recordings of real agent sessions (hours of footage)
- Separate video cutting software (Felipe's existing workflow)

## Distribution Strategy
- X (Twitter) — developer audience, short-form viral
- Reddit — r/programming, r/artificial, r/OpenSource
- LinkedIn — professional/enterprise angle
