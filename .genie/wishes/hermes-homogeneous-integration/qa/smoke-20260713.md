## Robustness checks (script behaves under failure + safety)

- **Idempotent re-run:** second `bash smoke.sh` against the same env →
  `symlink unchanged, mcp-config unchanged, skills-dir unchanged`; all checks PASS; exit 0.
- **Fails for the right reason:** with a tampered `plugin.yaml` (wrong version +

- **Precondition guard:** unset `GENIE_HOME` → `smoke: GENIE_HOME must be set …`, exit 2.
