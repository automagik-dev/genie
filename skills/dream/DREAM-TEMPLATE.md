# DREAM.md Template

This is the execution plan template generated by the `/dream` skill and written to `.genie/DREAM.md` in the shared worktree.

## Per-Wish Entry Fields

| Field | Value |
|-------|-------|
| `slug` | wish identifier |
| `branch` | `feat/<slug>` |
| `wish_path` | `.genie/wishes/<slug>/WISH.md` |
| `depends_on` | upstream slugs from WISH.md |
| `merge_order` | integer from topological layering |

## Dependency Layering

Wishes are assigned `merge_order` as integer layers `1..N`:

- **Layer 1**: wishes with no selected dependencies.
- **Increment layer**: when a wish depends on a lower layer.
- **Same-layer wishes**: are parallel and can be dispatched simultaneously.

The topological sort is computed across selected wishes using the `depends_on` field from each `WISH.md`.
